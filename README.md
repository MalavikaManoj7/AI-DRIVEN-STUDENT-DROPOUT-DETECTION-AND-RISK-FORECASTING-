# 🎓 AI Student Dropout Prediction

## Overview
Machine learning system to predict student dropout, enrolled, or graduate.

## Steps
1. Train model
2. Run Streamlit app

## Run

pip install -r requirements.txt  
python train.py  
streamlit run app.py  

## Features
- Random Forest Model
- 37 Features
- SHAP Explainability
- Probability Graph
